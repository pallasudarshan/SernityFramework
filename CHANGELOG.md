kyma-platform-mobile- - What's in this release
This document summarizes changes made to the kyma-platform-mobile test framework.

Version: 7.2.0

----------------------------------
### Changes

* New Test Framework for testing Kyma mobile widgets
* Added test scenarios and test code related to Patient Banner
* Added support for GooglePixel9ProXL15 and SamsungGalaxyS24Ultra14

Version: 7.3.0

----------------------------------
### Changes

* Improved Patient Banner test scenarios and code for Alarm widget and header location
* Added FSW waveform layout test scenarios and code
* Introduced Caliper widget test scenarios and code
* Enhanced gesture support in the test framework, including zoom, scroll, swipe, and image comparison etc
* Scoped Galaxy A52 and descoped Galaxy S24 Ultra due to unavailability of the real device
* Enabled WSS protocol testing on BrowserStack Appium cloud

Version: 7.4.0

----------------------------------
### Changes

* Introduced Numeric Block widget with Tile, Condensed and Combined views
* Implemented EWS Score widget with Tile, Combined and Badge views
* Wrapped Layout View has been added as an option for Patient Banner Expanded mode
* Patient Banner can now show either badges or alarms in the collapsed view and tap events on title,chevron button
* Enhanced gesture support in test framework like zoom effect, disabling scroll during zoom, Caliper branch proximity handling
* Scoped iPhone16Pro device with enablement of gitlab pipeline with BrowserStack integration for automated testing
